# DMX Lights
Forked from https://github.com/rydercalmdown/dmx_lights


Controlling a WKL-BEAM02 RGBW DMX pinspot from a Raspberry Pi with an Enttec Open DMX USB dongle.

Goal: set RGBW values via a simple web interface.

## Endpoints
- /wash/random/
- /wash/white/
- /wash/red/
- /wash/blue/
- /wash/green/
- /wash/yellow/
- /wash/purple/
- /wash/aqua/
- /blackout/
